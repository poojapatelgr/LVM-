How Remaining Personal/Sick Leave are calculated ?
Personal Leave = Rollover + (Number of months from Start Date * Personal Leave per month) + Max. Personal Leave(Borrow) - Taken

Sick Leave = (Number of months from Start Date * Sick Leaves per month) + Max. Sick Leave(Borrow) - Taken

Example 1:
Role: Consultant
Start Date = 01/01/2022
Rollover = 0
Personal Leave per month = 1.5
Sick Leave per month = 1
On Jan 1st, 2022:
Remaing Personal Leave = 0 + (0 * 1.5) + 3 = 3 days
Remaing Sick Leave = (0 * 1) + 3 = 3 days

Example 2:
Role: Consultant
Start Date = 01/01/2022
Rollover = 0
Earned Personal Leave per month = 1.5
Earned Sick Leave per month = 1.5
Personal Leave already taken = 2
On Feb 1st, 2022:
Remaing Personal Leave = 0 + (1 * 1.5) + 3 - 2 = 2.5 days
Remaing Sick Leave = (1 * 1) + 3 = 4 days
